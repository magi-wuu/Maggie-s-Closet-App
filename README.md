### Name
MAGGIE'S CLOSET

### Description
MAGGIE'S CLOSET is an interactive program developed using Pygame, which allows users to match and create outfits. Users can select tops, bottoms, hair, and shoes for a model, score the outfits, and save their favorite 4 combinations.

### Features
- **Interactive Interface**: Allows users to choose from various tops, bottoms, hair, and shoes.
- **Scoring System**: Provides scores based on outfit compatibility and predefined rules.
- **Background Music**: Plays background music during the game for an engaging experience.
- **Save and View Outfits**: Users can save their created outfits and view them in a gallery.
- **Scrolling Background:** A smooth scrolling background in the outfit gallery.
